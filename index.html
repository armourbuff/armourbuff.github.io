🔧 How to update my résumé

Pick whichever is easier — you never have to touch the code in either case:

A

Replace the file (simplest)

Keep your PDF named exactly Amit_Kumar_Gupta_CV.pdf and place it next to index.html. To update later, overwrite it with a new PDF of the same name. On GitHub: Add file → Upload files → commit.

B

Use a Drive/Dropbox link (zero file management)

Host the PDF on Google Drive or Dropbox once, then paste its share link into the RESUME.url line near the top of the page's script. After that, updating your résumé = replacing the file in Drive. The site link never changes.

💡 Also update RESUME.updated in the same spot to refresh the "Last updated" date shown above. After any change, hard-refresh (Ctrl/Cmd+Shift+R) to clear the cached PDF.
